--- title: Window Proxy slug: Glossary/Window_Proxy tags: - Glossary - Window - Window Proxy ---

The Window Proxy object is a wrapper for the Window object. It exists in every browsing context. All operations performed on a Window Proxy object will also be applied to its underlying wrapped Window object. Therefore, interacting with the Window Proxy object is almost identical to interacting with the Window object until the browsing context is navigated.
