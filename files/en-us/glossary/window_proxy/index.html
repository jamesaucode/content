--- title: Window Proxy slug: Glossary/Window_Proxy tags: - Glossary - Window - Window Proxy ---

A Window Proxy is a wrapper for the Window object. It exist in every Browsing Context. All operations performed on the Window Proxy will also be applied to it's underlying wrapped Window object. Therefore, interacting with the Window Proxy object is almost identical with interacting with the Window object until the Browsing Context is navigated.
